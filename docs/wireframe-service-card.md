# Wireframe: Карточка услуги — Калды.ру

> Документ описывает структуру и содержимое wireframe страницы карточки услуги.
> Используется как основа для дизайна в Figma.
> Все решения приняты на основе `brief.md`, `content.md` и `wireframe-services-catalog.md`.

---

## Структура страницы (сверху вниз)

### 1. Navbar
Идентичен каталогу услуг.

**Содержимое:**
- Логотип / название «Калды.ру» — слева
- Навигационные ссылки по центру: Номера · **Услуги** (активный, подчёркнут `#1A6B8A`) · Галерея · Как добраться
- CTA-кнопка «Забронировать» — справа (цвет: акцент `#E8A838`)

---

### 2. Хлебные крошки

`Главная › Услуги › Баня на берегу озера`

- «Главная» и «Услуги» — кликабельные ссылки
- Название услуги — текущая страница (не ссылка)

---

### 3. Основной контент — двухколоночная раскладка

**Пропорции колонок:** левая ~62%, правая ~38%

---

#### Левая колонка (контент)

**Заголовок и теги:**
- H1: название услуги (например, «Баня на берегу озера»)
- Теги-чипы: категория (`#1A6B8A`) · «Популярная» (`#E8A838`, если featured) · вместимость (серый)

**Фото:**
- Схема: главное фото (большое, слева) + 2 миниатюры (справа, стопкой)
- Кнопка «+N фото» поверх последней миниатюры → лайтбокс
- Высота главного фото: ~140px
- Примечание: у большинства услуг фото меньше, чем у номеров — галерея проще

**Описание:**
- Раздел H3 «Описание»
- 2–4 предложения об услуге

**Подробности:**
- Раздел H3 «Подробности»
- Таблица key-value с разделителями:

| Параметр | Пример значения |
|----------|----------------|
| Вместимость | до 6 человек |
| Тип | Баня на дровах |
| Расположение | На берегу озера |
| В комплекте | Комната отдыха, дорожка до воды |
| Режим работы | По согласованию |

**Правила использования:**
- Раздел H3 «Правила использования»
- Список с точками `#1A6B8A`:
  - Бронирование заранее — у администратора
  - Минимальное время аренды — 1 час
  - Оплата только наличными
- Предупреждающие пункты — точка `#E8A838`, текст `#7A5000`

---

#### Правая колонка (информационный блок)

**Поведение:** `position: sticky; top: 12px` — прилипает при скролле на десктопе.

**Отличие от карточки номера:** нет кнопки «Забронировать» — услуги бронируются только по телефону. Главный CTA — «Узнать доступность».

**Содержимое блока:**

| Элемент | Детали |
|---------|--------|
| Цена | Крупно, `#1A6B8A`, 22px |
| Подпись к цене | «Минимум 1 час» / «за человека» и т.д. |
| Параметры | Чипы: вместимость, тип, особенность |
| Разделитель | `0.5px` |
| CTA «Узнать доступность» | Кнопка акцент `#E8A838`, 100% ширина |
| CTA «Позвонить нам» | Кнопка контур `#1A6B8A`, 100% ширина |
| Телефон + часы | +7-922-707-02-44, с 9:00 до 21:00 |
| Разделитель | `0.5px` |
| «Другие услуги рядом» | Компактный список: название + цена (3 позиции) |
| Разделитель | `0.5px` |
| Шеринг | Три кнопки: VK · WA · TG |

**Логика «Других услуг рядом»:** показывать услуги из той же категории или логически связанные (к бане — мангал, к лодкам — катамаран). Стимулирует докупить смежные услуги.

---

### 4. Блок «Похожие услуги»

**Тип:** горизонтальная строка из 3 тайлов

**Структура мини-тайла:**
- Фото (~55px)
- Название
- Цена (или «уточнить»)

**CTA внизу блока:** «Все услуги →» (контурная, серая)

---

### 5. Footer
Идентичен остальным страницам.

## Отличия от карточки номера

| Параметр | Карточка номера | Карточка услуги |
|----------|----------------|----------------|
| Главный CTA | «Забронировать» | «Узнать доступность» |
| Вторичный CTA | «Позвонить нам» | «Позвонить нам» |
| Правая колонка | Цена + бронирование | Цена + контакты + «Другие услуги рядом» |
| Раздел деталей | «Удобства» (список с иконками) | «Подробности» (таблица key-value) |
| Галерея | 2–3 фото + миниатюры | 1–2 фото + миниатюры (проще) |
| Пагинация/навигация | Похожие номера | Похожие услуги |

---

## Мобильная версия (< 768px)

| Элемент | Решение |
|---------|---------|
| Двухколоночная раскладка | Одна колонка — контент сверху, блок справа — снизу |
| Стики-блок | Фиксированная панель внизу экрана: цена + кнопка «Узнать доступность» |
| Галерея | Горизонтальный скролл |
| «Другие услуги рядом» | Переезжает в основной контент, перед блоком «Похожие услуги» |

| Тип услуги | Формат цены | Подпись |

## Дизайн-решения для Figma

### Цвета
| Роль | Hex |
|------|-----|
| Основной | `#1A6B8A` |
| Акцент / CTA | `#E8A838` |
| Фон страницы | `#F5F9FB` |
| Текст | `#1A2C35` |
| Предупреждение (наличные, советы) | `#E8A838` точка + `#7A5000` текст |

### Типографика
| Элемент | Шрифт | Размер |
|---------|-------|--------|
| H1 (название услуги) | Playfair Display | 28–32px |
| H3 (разделы) | Playfair Display | 18–20px |
| Цена основная | Inter / Nunito | 22px, weight 500, `#1A6B8A` |
| Подпись к цене | Inter / Nunito | 12px, secondary |
| Детали (key-value) | Inter | 13px |
| Правила | Inter | 12px, secondary |
| Мелкий текст (телефон, часы) | Inter | 12px |

### Компоненты для Figma

- [ ] Таблица key-value (параметры услуги)
- [ ] Список правил с точками (синяя / янтарная)
- [ ] Информационный блок правой колонки (отличается от карточки номера)
- [ ] Строка «Другие услуги рядом» (название + цена)
- [ ] Мини-тайл «Похожие услуги»
- [ ] Фиксированный футер-CTA (mobile): цена + кнопка
- [ ] Метка цены с вариантами подписи (час / день / чел. / бесплатно / уточнить)
- [ ] Чипы: категория синий / featured янтарный / серый
