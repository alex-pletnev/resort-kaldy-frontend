# Design Tokens — Калды.ру

> Все решения по визуальному стилю принимай на основе этих токенов.

---

## Логотип

**Файл:** `kaldy-logo.png`  
**Описание:** закат над озером — ели, солнце, волны воды, растение. Иконка + текст «Калды.ру».  
**Использование:**
- Навбар: логотип слева, высота ~36px, на светлом фоне
- Футер: логотип слева, высота ~32px, на тёмном фоне (`bg-dark`)
- Фавикон: только иконка (без текста), 32×32px

**Фон логотипа:** чёрный — при использовании на светлом фоне применять `mix-blend-mode: multiply` или использовать версию с прозрачным фоном.

> ⚠️ Текущий файл имеет чёрный фон. Нужна версия с прозрачным фоном (PNG с альфа-каналом) для корректного отображения в навбаре.

### Размеры (mobile → desktop)

| Элемент | Мобайл | Десктоп |
|---|---|---|
| Hero заголовок | 2.5rem | 3.5rem |
| Заголовок секции | 2rem | 2.5rem |
| Заголовок карточки | 1.25rem | 1.5rem |
| Основной текст | 1.0625rem (17px) | 1.0625rem |
| Мелкий текст | 0.875rem | 0.875rem |

> Размер основного текста 17px (вместо стандартных 16px) — для удобства аудитории 40–70 лет.

## Отступы

Используй стандартную шкалу Tailwind (`p-4` = 16px, `p-8` = 32px, `p-16` = 64px).

| Использование | Tailwind класс | Размер |
|---|---|---|
| Отступ секции по Y | `py-16` / `py-20` | 64–80px |
| Отступ контейнера по X | `px-6` / `px-8` | 24–32px |
| Отступ внутри карточки | `p-5` / `p-6` | 20–24px |
| Зазор между карточками | `gap-4` / `gap-6` | 16–24px |
| Отступ между секциями | `mb-12` / `mb-16` | 48–64px |

## Кнопки

**Первичная (CTA):**
- Фон: `accent` (#E8A838)
- Текст: `bg-dark` (#1A2C35)
- Hover: `accent-dark`
- Форма: `rounded-full`, `px-8 py-3`

**Вторичная:**
- Фон: прозрачный
- Рамка: `primary`
- Текст: `primary`
- Hover: фон `bg-secondary`

**На тёмном фоне (hero):**
- Фон: белый
- Текст: `primary`
- Hover: `bg-secondary`

## Подключение шрифтов (layout.tsx)

```tsx
import { Playfair_Display, Nunito } from 'next/font/google'

const playfair = Playfair_Display({
  subsets: ['latin', 'cyrillic'],
  variable: '--font-playfair',
  display: 'swap',
})

const nunito = Nunito({
  subsets: ['latin', 'cyrillic'],
  variable: '--font-nunito',
  display: 'swap',
})
```
