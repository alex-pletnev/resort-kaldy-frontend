# Wireframe: Документы и правила — Калды.ру

> Документ описывает структуру и содержимое wireframe страницы «Документы».
> Используется как основа для дизайна в Figma.
> Все решения приняты на основе `brief.md` и `content.md`.

---

## Структура страницы (сверху вниз)

### 1. Navbar
Идентичен остальным страницам.

**Содержимое:**
- Логотип / название «Калды.ру» — слева
- Навигационные ссылки по центру: Номера · Услуги · Галерея · Как добраться
- CTA-кнопка «Забронировать» — справа (цвет: акцент `#E8A838`)

**Примечание:** «Документы» нет в основной навигации — страница доступна из Footer («Документы» · «Правила»).

---

### 2. Шапка страницы

**Содержимое:**
- Хлебные крошки: `Главная › Документы`
- Заголовок H1: «Документы и правила»
- Подзаголовок: «Официальная информация, правила проживания и поведения на базе»

---

### 3. Двухколоночная раскладка — сайдбар + контент

**Пропорции:** левая (навигация) ~160px фиксированная, правая (контент) — остаток

---

#### Левая колонка — боковая навигация (sticky)

**Поведение:** `position: sticky; top: 12px`

**Фон:** `var(--color-background-secondary)`, `border-radius: var(--border-radius-lg)`

**Разделы:**
1. Документы *(активный)*
2. Правила проживания
3. Правила поведения
4. Реквизиты

**Активный раздел:** фон `#1A6B8A15`, цвет `#1A6B8A`, font-weight 500

**Внизу сайдбара:** мелкая подсказка «Вопросы? Позвоните нам» со ссылкой-телефоном

**Решение:** для аудитории 40–70 лет сайдбар удобнее якорных ссылок в тексте — структура видна сразу, переход к нужному разделу очевиден.

---

#### Правая колонка — контент

---

##### Раздел 1: Официальные документы

Список карточек документов, одна под другой.

**Структура карточки документа:**
- Иконка файла (36×44px): PDF — фон `#FAEEDA`, загнутый уголок `#E8A838`; TXT/HTML — фон `#E6F1FB`, уголок `#1A6B8A`
- Название документа (font-weight 500)
- Краткое описание / реквизиты (muted, 11px)
- Кнопки: «Открыть» · «Скачать» (ghost)

**Документы:**

| Документ | Тип | Описание |
|----------|-----|----------|
| Свидетельство о регистрации ИП | PDF | ИП Габринец Лариса Юрьевна · ОГРНИП 326745600047589 |
| Договор оферты | PDF | Условия оказания услуг базы отдыха |
| Политика конфиденциальности | PDF | Обработка персональных данных |

---

##### Раздел 2: Правила проживания

Три смысловых блока, каждый с заголовком и нумерованным списком.

**Блок «Заезд и выезд»** + кнопка «Скачать PDF»:
1. Заезд с 14:00. Ранний заезд — по согласованию с администратором.
2. Выезд до 12:00. Поздний выезд — по согласованию, может быть платным.
3. При заселении предъявить документ, удостоверяющий личность.
4. Оплата только наличными при заселении.

**Блок «Проживание»**:
1. Курение — только в специально отведённых местах.
2. Тихий час с 23:00 до 07:00.
3. Животные — по согласованию, доп. оплата 500 ₽/сутки.
4. Ответственность за сохранность имущества несёт проживающий.

---

##### Раздел 3: Правила поведения на территории

**Блок «Правила поведения»**:
1. Соблюдать чистоту на территории и в номерах.
2. Не шуметь после 23:00.
3. Купание только в разрешённых местах.
4. Разведение огня — только в мангальных зонах.

---

##### Раздел 4: Реквизиты организации

**Тип:** key-value сетка 2×2

| Поле | Значение |

### 4. Footer
Идентичен остальным страницам. В Footer ссылки «Документы» и «Правила» ведут на эту страницу (якоря к разным разделам).

## Дизайн-решения

**Страница намеренно сдержанная** — это справочная/юридическая информация, не продающая. CTA-блока внизу нет.

**Иконки типов файлов:**
- PDF: фон `#FAEEDA`, загнутый уголок `#E8A838`, текст «PDF» `#8B5E00`
- TXT/HTML: фон `#E6F1FB`, загнутый уголок `#1A6B8A`, текст «TXT» `#0D4A60`

**Нумерованные пункты правил:**
- Кружок с номером: фон `#1A6B8A15`, цвет `#1A6B8A`, 18×18px
- Разделитель между пунктами: `0.5px solid var(--color-border-tertiary)`

## Мобильная версия (< 768px)

| Элемент | Решение |
|---------|---------|
| Двухколоночная раскладка | Одна колонка |
| Боковая навигация | Горизонтальные чипы-якоря под заголовком (скролл) |
| Карточки документов | Полная ширина |
| Блоки правил | Полная ширина |

---

## Типографика для Figma

| Элемент | Шрифт | Размер |
|---------|-------|--------|
| H1 | Playfair Display | 32px |
| H2 (разделы) | Inter / Nunito | 14px, weight 500 |
| H3 (подгруппы правил) | Inter / Nunito | 12px, weight 500 |
| Название документа | Inter | 13px, weight 500 |
| Пункты правил | Inter | 12px, secondary |
| Реквизиты — подпись | Inter | 11px, tertiary |
| Реквизиты — значение | Inter | 12px, weight 500 |

## Компоненты для Figma

- [ ] Карточка документа (иконка PDF / TXT + название + кнопки)
- [ ] Иконка файла (PDF-вариант, TXT-вариант)
- [ ] Блок правил (заголовок + нумерованный список + опциональная кнопка PDF)
- [ ] Пункт правила (кружок с номером + текст)
- [ ] Боковая навигация сайдбара (активный / неактивный пункт)
- [ ] Сетка реквизитов (key-value 2×2)
- [ ] Кнопки: ghost «Открыть» / «Скачать», ghost «Скачать PDF»
