# Wireframe: Карточка номера — Калды.ру

> Документ описывает структуру и содержимое wireframe страницы карточки номера.
> Используется как основа для дизайна в Figma.
> Все решения приняты на основе `brief.md`, `content.md` и `wireframe-catalog.md`.

---

## Структура страницы (сверху вниз)

### 1. Navbar
Идентичен каталогу номеров.

**Содержимое:**
- Логотип / название «Калды.ру» — слева
- Навигационные ссылки по центру: **Номера** (активный, подчёркнут `#1A6B8A`) · Услуги · Галерея · Как добраться
- CTA-кнопка «Забронировать» — справа (цвет: акцент `#E8A838`)

---

### 2. Хлебные крошки

`Главная › Номера и коттеджи › Коттедж №7 / №8`

- «Главная» и «Номера и коттеджи» — кликабельные ссылки
- Название номера — текущая страница (не ссылка)

---

### 3. Основной контент — двухколоночная раскладка

**Пропорции колонок:** левая ~62%, правая ~38%

---

#### Левая колонка (контент)

**Заголовок и теги:**
- H1: название номера / коттеджа (например, «Коттедж №7 / №8»)
- Теги-чипы: «Популярный» (`#E8A838`) · вместимость (`#1A6B8A`) · тип жилья (серый)

**Галерея:**
- Схема: главное фото (большое, слева) + 2 фото меньшего размера (справа, стопкой)
- Кнопка «+N фото» поверх последнего миниатюрного фото → открывает лайтбокс
- Высота главного фото: ~200px
- Примечание: если фотографий 8+, рассмотреть лайтбокс с миниатюрами снизу

**Описание:**
- Раздел с H3 «Описание»
- Текстовый блок 3–5 предложений о номере

**Удобства:**
- Раздел с H3 «Удобства»
- Две колонки, список с точками цвета `#1A6B8A`
- Неизвестные удобства (Wi-Fi, ТВ, холодильник) помечаются «уточнить» — `#E8A838`
- ⚠️ Уточнить у клиента до дизайна: Wi-Fi, ТВ, холодильник, санузел

**Правила проживания:**
- Раздел с H3 «Правила проживания»
- Чипы: «Заезд с 14:00» · «Выезд до 12:00» · «Животные: +500 ₽/сут» · «Только наличные» (предупреждающий)

---

#### Правая колонка (стики-блок бронирования)

**Поведение:** `position: sticky; top: 12px` — прилипает при скролле на десктопе.

**Содержимое блока:**

| Элемент | Детали |
|---------|--------|
| Цена (будни) | Крупно, `#1A6B8A`, 22px |
| Цена (выходные) | Мелко, серый текст под основной ценой |
| Допместо | Мелко, серый текст |
| Вместимость | Чипы: основная + допместа |
| Тип жилья | Чип серый |
| Разделитель | `0.5px` |
| CTA «Забронировать» | Кнопка акцент `#E8A838`, 100% ширина |
| CTA «Позвонить нам» | Кнопка контур `#1A6B8A`, 100% ширина |
| Телефон + часы | +7-922-707-02-44, с 9:00 до 21:00, мелкий текст по центру |
| Разделитель | `0.5px` |
| Шеринг | Три кнопки: VK · WA · TG |

**Логика кнопки «Забронировать»:** сейчас → переход к контактам / звонку. В будущем → онлайн-форма с датами (заложить в архитектуру).

---

### 4. Блок «Похожие номера»

**Тип:** горизонтальная строка из 3 карточек

**Структура мини-карточки:**
- Фото (~60px)
- Название
- Цена от X ₽/сут

**CTA внизу блока:** кнопка «Все номера и коттеджи →» (контурная, серая)

**Назначение:** снизить отказы — если номер не подошёл, гость видит альтернативы без возврата в каталог.

---

### 5. Footer
Идентичен каталогу номеров.

## Мобильная версия (< 768px)

| Элемент | Решение |
|---------|---------|
| Двухколоночная раскладка | Одна колонка — контент сверху, блок бронирования снизу |
| Стики-блок бронирования | Фиксированная панель внизу экрана («липкий футер»): цена + кнопка «Забронировать» |
| Галерея | Горизонтальный скролл миниатюр |
| Удобства | Одна колонка вместо двух |

## Дизайн-решения для Figma

### Цвета
| Роль | Hex |
|------|-----|
| Основной | `#1A6B8A` |
| Акцент / CTA | `#E8A838` |
| Фон страницы | `#F5F9FB` |
| Текст | `#1A2C35` |
| Предупреждение (наличные, уточнить) | `#E8A838` светлый фон |

### Типографика
| Элемент | Шрифт | Размер |
|---------|-------|--------|
| H1 (название номера) | Playfair Display | 28–32px |
| H3 (разделы) | Playfair Display | 18–20px |
| Цена основная | Inter / Nunito | 22px, weight 500, `#1A6B8A` |
| Цена вторичная | Inter / Nunito | 13px, `#1A2C35` muted |
| Удобства, чипы | Inter | min. 14px |
| Мелкий текст (телефон, часы) | Inter | 12px |

### Компоненты для Figma

- [ ] Галерея: главное фото + 2 миниатюры + кнопка «+N фото»
- [ ] Лайтбокс (состояние: открыт)
- [ ] Блок удобств (список с точками, две колонки)
- [ ] Стики-блок бронирования (desktop)
- [ ] Фиксированный футер-CTA (mobile)
- [ ] Мини-карточка «Похожие номера»
- [ ] Чипы: акцент / синий / серый / предупреждение
- [ ] Кнопки: primary `#E8A838`, outline `#1A6B8A`
