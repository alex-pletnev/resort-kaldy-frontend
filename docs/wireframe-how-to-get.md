# Wireframe: Как добраться — Калды.ру

> Документ описывает структуру и содержимое wireframe страницы «Как добраться».
> Используется как основа для дизайна в Figma.
> Все решения приняты на основе `brief.md` и `content.md`.

---

## Структура страницы (сверху вниз)

### 1. Navbar
Идентичен остальным страницам.

**Содержимое:**
- Логотип / название «Калды.ру» — слева
- Навигационные ссылки по центру: Номера · Услуги · Галерея · **Как добраться** (активный, подчёркнут `#1A6B8A`)
- CTA-кнопка «Забронировать» — справа (цвет: акцент `#E8A838`)

---

### 2. Шапка страницы

**Содержимое:**
- Хлебные крошки: `Главная › Как добраться`
- Заголовок H1: «Как добраться»
- Подзаголовок: «Кунашакский район, Челябинская область · 62-й км трассы Челябинск — Екатеринбург»

---

### 3. Три ключевые метрики

**Тип:** три карточки-метрики в ряд (`background: var(--color-background-secondary)`)

| Карточка | Подпись | Значение | Примечание |
|----------|---------|----------|------------|
| 1 | от Челябинска | ~60 км | ~1 час на авто |
| 2 | поворот | 61-й км | трасса Чел. — Екб |
| 3 | от поворота | 300 м | по указателю |

**Решение:** самое важное сразу — без чтения. Критично для аудитории 40–70 лет, которая открывает страницу в дороге или при планировании.

---

### 4. Карта + маршруты (двухколоночная раскладка)

**Пропорции:** левая (карта) ~60%, правая (маршруты) ~40%

---

#### Левая колонка — карта

- Встроенная карта: Яндекс.Карты (приоритет) или Google Maps
- Маркер базы на карте — акцентный цвет `#E8A838`
- Высота карты: ~240px
- Под картой: две кнопки-ссылки
  - «Открыть в Яндекс.Картах» → внешняя ссылка
  - «Открыть в Google Maps» → внешняя ссылка

**Примечание:** у целевой аудитории чаще Яндекс.Карты, но оба варианта нужны.

---

#### Правая колонка — карточки маршрутов

**Три карточки, одна под другой:**

**1. На автомобиле** (основной маршрут)
- Иконка авто + заголовок + чип «~1 час»
- Пошаговые инструкции с нумерованными шагами:
  1. Выезд из Челябинска по Свердловскому тракту
  2. Трасса Челябинск — Екатеринбург, до 61-го км (~60 км)
  3. Поворот по указателю «База Калды» — 300 м *(шаг выделен акцентом `#E8A838`)*
- Шаги соединены вертикальной линией `#1A6B8A40`

**2. На автобусе**
- Иконка автобуса + заголовок
- Текстом: автобусы в направлении Екатеринбурга, Перми или Кунашака от автовокзала Челябинска. Выйти на «61-й км», 300 м по указателю.

**3. Трансфер**
- Иконка человека + заголовок + чип «по запросу»
- Текстом: по согласованию с администратором при бронировании
- Кнопка «Узнать о трансфере» → контакты (контурная `#1A6B8A`)

---

### 5. Адрес и контакты (двухколоночная раскладка)

**Левая часть — адрес:**
- Название региона: «Челябинская область»
- Строки адреса: Кунашакский район, оз. Калды; 62-й км трассы Челябинск — Екатеринбург
- Кнопки: «Скопировать адрес» · «Поделиться маршрутом» (ghost)

**Правая часть — контакты:**
- Телефон: +7-922-707-02-44 · с 9:00 до 21:00, без выходных
- Мессенджеры: WhatsApp / Telegram · тот же номер
- Режим работы: Сезон: май — сентябрь

---

### 6. Ориентиры в дороге

**Тип:** список с точками `#1A6B8A`

- 49 км от границы Челябинска — ориентир для навигатора
- Знак «61 км» на трассе — поворот направо по указателю
- Берёзовый лес и указатель «База отдыха Калды» слева

**Полезные ссылки:**
- «Погода на gismeteo.ru» → https://www.gismeteo.ru/weather-suleymanovo-234768/
- «Отзывы ВКонтакте» → https://vk.com/kaldy_ru

**Решение:** бытовые подсказки важнее точных координат — навигаторы иногда путают грунтовки в этом районе.

---

### 7. CTA-блок

**Тип:** акцентный блок с лёгким фоном `#1A6B8A12`

**Содержимое:**
- Заголовок: «Уже знаете дорогу?»
- Подзаголовок: «Позвоните нам или напишите — поможем с выбором номера»
- Кнопки: «Забронировать» (`#E8A838`) + «Позвонить нам» (контур `#1A6B8A`)

**Логика:** страница «Как добраться» — финальный шаг перед принятием решения. CTA здесь конвертирует заинтересованных.

---

Идентичен остальным страницам.

## Мобильная версия (< 768px)

| Элемент | Решение |
|---------|---------|
| Метрики | 3 карточки в ряд (компактные) |
| Карта + маршруты | Одна колонка: карта сверху, маршруты снизу |
| Карта | Высота уменьшается до ~180px |
| Адрес + контакты | Одна колонка |
| Кнопки карт | Полная ширина, одна под другой |

## Дизайн-решения для Figma

### Цвета
| Роль | Hex |
|------|-----|
| Основной | `#1A6B8A` |
| Акцент / маркер карты / шаг 3 | `#E8A838` |
| Фон метрик | `var(--color-background-secondary)` |
| Фон CTA-блока | `#1A6B8A12` |
| Линия шагов маршрута | `#1A6B8A40` |
| Текст | `#1A2C35` |

### Типографика
| Элемент | Шрифт | Размер |
|---------|-------|--------|
| H1 | Playfair Display | 32px |
| Значение метрики | Inter / Nunito | 18px, weight 500, `#1A6B8A` |
| Заголовки карточек маршрутов | Inter | 13px, weight 500 |
| Шаги маршрута | Inter | 12px, secondary |
| Адрес, контакты | Inter | 12–13px |
| Ориентиры | Inter | 12px, secondary |

### Компоненты для Figma

- [ ] Карточка-метрика (значение + подпись + примечание)
- [ ] Карточка маршрута: авто (со шагами) / автобус / трансфер
- [ ] Шаг маршрута с номером и соединительной линией
- [ ] Блок адреса с кнопками
- [ ] Строка контакта (иконка + название + подпись)
- [ ] CTA-блок с двумя кнопками
- [ ] Кнопки: «Открыть в картах» (ghost), primary `#E8A838`, outline `#1A6B8A`
