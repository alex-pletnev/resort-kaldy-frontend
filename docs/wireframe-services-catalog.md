# Wireframe: Каталог услуг — Калды.ру

> Документ описывает структуру и содержимое wireframe страницы каталога услуг.
> Используется как основа для дизайна в Figma.
> Все решения приняты на основе `brief.md`, `content.md` и `wireframe-catalog.md`.

---

## Структура страницы (сверху вниз)

### 1. Navbar
Идентичен остальным страницам.

**Содержимое:**
- Логотип / название «Калды.ру» — слева
- Навигационные ссылки по центру: Номера · **Услуги** (активный, подчёркнут `#1A6B8A`) · Галерея · Как добраться
- CTA-кнопка «Забронировать» — справа (цвет: акцент `#E8A838`)

---

### 2. Шапка страницы

**Содержимое:**
- Хлебные крошки: `Главная › Услуги`
- Заголовок H1: «Услуги и развлечения»
- Подзаголовок: «Всё для отдыха на воде, в лесу и у костра»

---

### 3. Панель фильтров + сортировка
**Тип:** горизонтальная панель
**Фон:** `#F5F9FB`

**Фильтр по категориям (чипы):**

| Чип | Категория |
|-----|-----------|
| Все | Показать все услуги |
| Водный отдых | Лодки, катамаран, пляж |
| Баня и отдых | Бани, мангальные зоны |
| Питание | Столовая, питание |
| Активности | Волейбол, игровые зоны |
| Территория | Вход, парковка |

**Сортировка:** выпадающий список — По цене ↑ · По цене ↓

**UX-детали:**
- Активный чип: фон `#E8A83825`, цвет текста `#7A5000`, рамка `#E8A83870`
- При выборе категории — либо прокрутка к блоку, либо скрытие остальных (решить на этапе разработки)
- Фильтры применяются мгновенно

---

### 4. Строка результатов + переключение вида

- Слева: «Найдено услуг: N»
- Справа: ⊞ Сетка · ≡ Список

---

### 5. Тематические группы с тайлами

**Ключевое решение:** услуги сгруппированы по категориям с заголовком группы, а не просто сеткой.
Причина: аудитория 40–70 лет — визуальная группировка удобнее фильтрации.

**Структура каждой группы:**
- Иконка категории + заголовок H2
- Сетка тайлов: 3 колонки на десктопе

### 6. Структура тайла услуги

**Состояния:**

| Состояние | Оформление |
|-----------|------------|
| Обычный | Белый фон, контурная кнопка `#1A6B8A` |
| Featured (популярная) | Янтарная рамка `#E8A838` 1.5px, бейдж «Популярная», акцентная кнопка |
| Hover | Лёгкая тень, кнопка меняет фон |

**Содержимое тайла (сверху вниз):**
1. Фото (~80px высота)
2. Название услуги
3. Подпись: категория · вместимость (если есть)
4. Разделитель
5. Строка: подпись «от» / «цена» + значение цены
6. Кнопка «Подробнее»

**Цвет цены:**
- Платная: `#1A6B8A`, weight 500
- Бесплатная: более светлый синий `#0D4A60` — визуально отличается от платной

## Отличия от каталога номеров

| Параметр | Каталог номеров | Каталог услуг |
|----------|----------------|---------------|
| Структура | Плоская сетка | Сетка с группами по категориям |
| Фильтр | Тип, вместимость, цена | Только категория |
| Сортировка | По цене, популярности | По цене |
| Пагинация | Есть | Нет (услуг мало, всё на одной странице) |
| Цена «бесплатно» | Не применимо | Особый стиль — отличается от платной |

---

## Мобильная версия (< 768px)

| Элемент | Решение |
|---------|---------|
| Фильтр-чипы | Горизонтальный скролл в одну строку |
| Сетка тайлов | 2 колонки (не 1 — тайлы услуг компактнее номеров) |
| Заголовки групп | Остаются, помогают ориентироваться |

## Дизайн-решения для Figma

### Цвета
| Роль | Hex |
|------|-----|
| Основной | `#1A6B8A` |
| Акцент / Featured | `#E8A838` |
| Фон страницы | `#F5F9FB` |
| Цена платная | `#1A6B8A` |
| Цена бесплатная | `#0D4A60` (светлее) |
| Текст | `#1A2C35` |

### Типографика
| Элемент | Шрифт | Размер |
|---------|-------|--------|
| H1 (заголовок страницы) | Playfair Display | 32px |
| H2 (заголовок группы) | Playfair Display | 18px |
| Название тайла | Inter / Nunito | 14px, weight 500 |
| Подпись (категория, вместимость) | Inter | 12px, tertiary |
| Цена | Inter | 13px, weight 500 |
| Текст фильтров | Inter | 14px, min. 16px |
