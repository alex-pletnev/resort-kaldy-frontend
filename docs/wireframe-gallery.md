# Wireframe: Фотогалерея — Калды.ру

> Документ описывает структуру и содержимое wireframe страницы фотогалереи.
> Используется как основа для дизайна в Figma.
> Все решения приняты на основе `brief.md` и `content.md`.

---

## Структура страницы (сверху вниз)

### 1. Navbar
Идентичен остальным страницам.

**Содержимое:**
- Логотип / название «Калды.ру» — слева
- Навигационные ссылки по центру: Номера · Услуги · **Галерея** (активный, подчёркнут `#1A6B8A`) · Как добраться
- CTA-кнопка «Забронировать» — справа (цвет: акцент `#E8A838`)

---

### 2. Шапка страницы

**Содержимое:**
- Хлебные крошки: `Главная › Галерея`
- Заголовок H1: «Фотогалерея»
- Подзаголовок: «Природа, территория, номера и отдых на озере Калды»

---

### 3. Фильтр по категориям + счётчик

**Тип:** горизонтальная панель с чипами
**Фон:** `#F5F9FB`

**Категории (чипы):**

| Чип | Что включает |
|-----|-------------|
| Все фото | Показать всё |
| Природа и озеро | Вода, лес, закаты, пейзажи |
| Территория | Корпуса, инфраструктура, баня, мангал |
| Номера | Интерьеры коттеджей и домиков |
| Пляж | Лежаки, понтон, купание, лодки |
| Мероприятия | Отдых с семьёй, дети, компании |

**Справа от чипов:** счётчик «Фото: N»

**UX-детали:**
- Активный чип: фон `#E8A83825`, цвет `#7A5000`, рамка `#E8A83870`
- Фильтр применяется мгновенно (без кнопки «Применить»)
- На мобильном: горизонтальный скролл в одну строку

---

### 4. Hero-блок — три фото крупным планом

**Тип:** 2 колонки — широкое фото слева (2/3), два фото стопкой справа (1/3)
**Высота:** ~200px

**Решение:** первые три фото задают атмосферу. Главный кадр — панорама озера или закат. Это важнее чёткой сетки — «продающий» вход в галерею.

**На каждом фото при hover:**
- Тёмная плашка снизу (`#1A6B8A`)
- Подпись категории (мелкий текст)
- Иконка-лупа с фоном `#E8A838` → открывает лайтбокс

---

### 5. Основная сетка — неравномерная (masonry-ритм)

**Тип:** 4 колонки с разными размерами ячеек
**Зазор между фото:** 3px

**Принципы сетки:**
- Горизонтальные фото (пейзажи, групповые) — на 2 колонки (`grid-column: span 2`)
- Вертикальные фото (портреты, домики) — на 2 строки (`grid-row: span 2`)
- Обычные — 1×1

**Решение:** неравномерная сетка живее однообразной, лучше подходит для природной тематики базы отдыха.

**Hover-подсказка на каждом фото:**
- Тёмная плашка снизу `#1A6B8A` (анимация: slide up)
- Название категории — мелкий текст `#B0D8E8`
- Иконка-лупа `#E8A838` → открывает лайтбокс на этом фото
- На мобильном: плашка показывается постоянно (нет hover)

**Кнопка «Показать ещё фото»** по центру снизу сетки:
- Ghost-кнопка
- Подгружает следующие N фото (lazy load)
- Исчезает когда все фото загружены

---

### 6. Лайтбокс (состояние: открыт)

**Фон:** тёмный оверлей `rgba(10, 25, 35, 0.88)`
**Закрытие:** кнопка ✕ в правом верхнем углу или клик вне фото

**Структура лайтбокса (сверху вниз):**

1. **Шапка:**
   - Слева: категория · счётчик «3 / 48»
   - Справа: кнопка поделиться + кнопка закрыть ✕

2. **Фото + стрелки:**
   - Активное фото по центру
   - Стрелка ← слева (круглая, полупрозрачная)
   - Стрелка → справа (круглая, полупрозрачная)
   - Навигация клавишами ← → на десктопе
   - Свайп на мобильном

3. **Полоса миниатюр внизу:**
   - Горизонтальный ряд миниатюр 36×26px
   - Активная миниатюра — рамка `#E8A838`
   - Счётчик `+N` если фото больше видимой области

---

### 7. CTA-блок

**Тип:** акцентный блок с фоном `#1A6B8A12`

**Содержимое:**
- Заголовок: «Понравилось? Приезжайте!»
- Подзаголовок: «Свяжитесь с нами и выберите удобный номер»
- Кнопки: «Смотреть номера» (`#E8A838`) + «Позвонить нам» (контур `#1A6B8A`)

**Логика:** галерея — эмоциональная страница, после просмотра фото человек готов к действию.

---

### 8. Footer
Идентичен остальным страницам.

## Категории фото — подробнее

| Категория | Что снимать | Приоритет |
|-----------|-------------|-----------|
| Природа и озеро | Вода, закаты, берёзовый лес, отражения | Высокий — главная «продающая» категория |
| Пляж | Лежаки, понтон, купание, лодки, дети у воды | Высокий |
| Территория | Корпуса снаружи, баня, мангал, дорожки, парковка | Средний |
| Номера | Интерьеры, кровати, виды из окна, террасы | Средний |
| Мероприятия | Семьи с детьми, барбекю, волейбол, вечерний отдых | Низкий |

## Мобильная версия (< 768px)

| Элемент | Решение |
|---------|---------|
| Фильтр-чипы | Горизонтальный скролл |
| Hero-блок | Одно фото на полную ширину |
| Основная сетка | 2 колонки, равные ячейки (masonry убирается) |
| Hover-плашка | Показывается постоянно (нет hover) |
| Лайтбокс | Полный экран, свайп влево/вправо |
| Миниатюры лайтбокса | Скрываются (экономия места) |

## Дизайн-решения для Figma

### Цвета
| Роль | Hex |
|------|-----|
| Основной | `#1A6B8A` |
| Акцент (иконка лупы, активная миниатюра, активный чип) | `#E8A838` |
| Hover-плашка на фото | `#1A6B8A` |
| Текст категории на плашке | `#B0D8E8` |
| Оверлей лайтбокса | `rgba(10, 25, 35, 0.88)` |
| Фон CTA-блока | `#1A6B8A12` |

### Типографика
| Элемент | Шрифт | Размер |
|---------|-------|--------|
| H1 | Playfair Display | 32px |
| Категория на плашке | Inter | 10px, `#B0D8E8` |
| Счётчик в шапке лайтбокса | Inter | 11px, `rgba(255,255,255,0.5)` |
| Подпись категории в лайтбоксе | Inter | 12px |

### Компоненты для Figma

- [ ] Фото-ячейка (состояния: обычная / hover / активная в лайтбоксе)
- [ ] Hover-плашка (категория + иконка лупы)
- [ ] Hero-блок 2+1 (три фото)
- [ ] Сетка 4 колонки с вариантами ячеек: 1×1, 2×1, 1×2
- [ ] Лайтбокс: оверлей + фото + стрелки + счётчик + миниатюры
- [ ] Миниатюра лайтбокса (обычная / активная `#E8A838`)
- [ ] Чип категории (активный / неактивный)
- [ ] CTA-блок с двумя кнопками
- [ ] Кнопки стрелок лайтбокса (круглые, полупрозрачные)
